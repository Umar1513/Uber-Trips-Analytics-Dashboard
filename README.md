# Uber Trip Analysis Dashboard

## Overview
Interactive Power BI dashboard built using a star schema data model to analyze 4,000 Uber trips. The report provides insights into revenue, trip patterns, driver performance, rider behavior, operations, and city-level trends.

## Dataset
- 4,000 trip records
- Fact_Trips table
- 7 dimension tables
- Excel source file

## Data Model
Star schema architecture with one fact table and multiple dimension tables.

## Dashboard Pages
1. Executive Dashboard
2. Driver Analysis
3. Rider Analysis
4. Operations Analysis
5. Geographic & City Analysis

## Key Metrics
- Total Trips
- Total Revenue
- Net Revenue
- Average Fare
- Average Duration
- Cancellation Rate
- Average Surge
- Average Rider Rating
